The Responsive Architecture

Every element is positioned with purpose. From the 60px nav container to the 360px hero section, every dimension follows the blueprint.

  • Container max-width 1080px with 20px gutters
  • Nav: sticky, 60px, backdrop blur for depth
  • Hero: full viewport height, centered grid
  • Cards: fluid grid, uniform shadow system
  • Footer: 4-column → stacked on mobile
